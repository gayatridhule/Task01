# 🌐 Responsive SaaS Landing Page

A fully responsive hero landing page built for a SaaS product using **HTML, CSS, and JavaScript**.  
This project focuses on mobile friendliness, a sticky navigation bar that changes color on scroll, and impactful typography.

---

## 🚀 Features
- Responsive design for all devices  
- Sticky navigation bar with scroll color change  
- Modern UI with gradient hero section  
- Smooth scrolling navigation  
- Clean and professional layout  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Font Awesome Icons
